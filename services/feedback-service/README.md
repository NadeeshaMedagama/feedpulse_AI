# Feedback Service

Owns feedback persistence, query/filter/pagination, status updates, and AI result storage.

## Endpoints
- `POST /feedback`
- `GET /feedback`
- `GET /feedback/:id`
- `PATCH /feedback/:id`
- `DELETE /feedback/:id`
- `GET /feedback/summary`
- `POST /feedback/:id/reanalyze`

## Run
```bash
npm install
npm run dev
```
